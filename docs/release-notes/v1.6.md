# LGRBZ v1.6 Release Notes

## Summary

v1.6 upgrades the Tax Centre with capital gains, discount placeholders, franking placeholders, tax reports and accountant export preparation.

## Added

- CGT summary engine
- Capital gains/losses summary
- Australian CGT discount placeholder engine
- Franking summary placeholder engine
- Tax optimiser suggestions
- Tax Report workspace
- Accountant Export workspace
- Tax export summary model
- Tax navigation links

## Known Limitations

- FIFO and CGT logic should be reviewed before real tax use.
- CGT discount eligibility is placeholder-only until buy/sell lot holding periods are fully tracked.
- Franking credits require real franking percentage fields.
- PDF/CSV export is not implemented yet.
