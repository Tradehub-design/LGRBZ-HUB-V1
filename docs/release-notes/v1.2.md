# LGRBZ v1.2 Release Notes

## Summary

v1.2 upgrades LGRBZ from a premium workspace interface into a stronger portfolio engine platform.

## Added

- Market valuation engine
- Enhanced holdings with market value
- Unrealised P/L
- Unrealised return %
- Portfolio valuation summary
- Performance metrics engine
- Income metrics engine
- Portfolio alerts engine
- Top movers engine
- Financial year summaries
- Recommendations engine
- Portfolio snapshot engine
- Data quality engine
- Equity curve engine
- Recharts chart components
- FIFO lot engine foundation
- FIFO disposal register
- Return metrics engine
- Portfolio replay engine

## Updated Pages

- Dashboard
- Holdings
- Transactions
- Analytics
- Dividends
- Dividend Forecast
- Tax Centre
- Reports
- Business Model
- Portfolio Health
- Settings

## Known Limitations

- Market prices are still seeded/mock values.
- Live price API integration is planned for v2.0.
- FIFO engine is foundational and should be reviewed further before relying on it for tax.
- XIRR, CAGR and benchmark comparison are planned for the next analytics milestone.
