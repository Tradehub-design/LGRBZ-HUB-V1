# LGRBZ v1.3 Release Notes

## Summary

v1.3 upgrades the visual and analytics experience of LGRBZ with premium charts, heatmaps, richer cards, better workspace components and polished investor dashboards.

## Added

- Premium chart card system
- Recharts equity curve, donut, bar and horizontal charts
- Portfolio heatmap
- Holding detail cards
- Transaction timeline
- Income calendar
- Risk radar
- Animated KPI cards
- Asset brand registry
- Premium filter bars
- Mobile density and scrollbar polish

## Updated

- Dashboard
- Holdings
- Transactions
- Analytics
- Dividends
- Portfolio Allocation
- Portfolio Health
- Performance Attribution

## Known Limitations

- Search bars are visual only; functional filtering is planned next.
- Chart data is still based on seeded portfolio engine output.
- Live market prices are planned for v2.0.
