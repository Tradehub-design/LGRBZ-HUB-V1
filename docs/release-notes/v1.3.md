# LGRBZ v1.3 Release Notes

## Summary

v1.3 upgrades the visual and analytics experience of LGRBZ with premium charts, richer dashboards, interactive search, heatmaps, insight panels and stronger workspace polish.

## Added

- Premium chart card system
- Recharts equity curve, donut, bar and horizontal charts
- Portfolio heatmap
- Holding detail cards
- Transaction timeline
- Income calendar
- Risk radar
- Animated KPI cards
- Asset brand registry
- Premium filter bars
- Functional holdings search
- Functional transactions search
- Command Centre widgets
- AI-style insight feed
- Dashboard widget shell
- Mobile density and scrollbar polish

## Updated

- Dashboard
- Holdings
- Transactions
- Analytics
- Dividends
- Portfolio Allocation
- Portfolio Health
- Performance Attribution

## Known Limitations

- Search is local browser-side filtering only.
- Live market prices are still planned for v2.0.
- AI insights are generated from deterministic portfolio rules, not a live LLM yet.
- Dashboard widgets are not drag-and-drop yet.
