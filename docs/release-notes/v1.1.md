# LGRBZ v1.1 Release Notes

## Summary

v1.1 upgrades LGRBZ from a placeholder dashboard into a professional portfolio operating system foundation.

## Major Changes

- Professional dark app shell
- Permanent sidebar navigation
- Premium dashboard visual system
- Seeded transaction CSV loaded from `/public/data/transactions.csv`
- Transaction engine as source of truth
- Holdings from transaction engine
- Allocation from transaction engine
- Dividend history from transaction engine
- Dividend forecast base
- Portfolio health scoring
- Portfolio risk scoring
- Analytics workspace
- Performance attribution workspace
- Tax Centre workspace
- Reports workspace
- Goals workspace
- Watchlist workspace
- Business Model workspace
- Broker Sync placeholder
- Live Prices placeholder
- Settings workspace
- Premium company-style asset logo bubbles
- Premium cards, badges, tables and status components

## Known Limitations

- Live prices are not connected yet.
- Unrealised P/L uses cost-basis mode until market prices are added.
- Tax Centre is event-based only; FIFO CGT engine is planned for v1.2.
- Broker sync is planned for v2.0.
- Report export to PDF/Excel is planned for a later version.
